# Запуск автотестов

> Написано на фреймворке Playwright на TypeScript

### 1. Необходимо установить или обновить npm для загрузки необходимого фреймворка

```npm install -g npm@11.12.1```

### 2. Необходимо установить фреймворк для автотестирования

```npm init playwright@latest```

### 3. Запуск тестов

```npx playwright test```
